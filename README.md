# Motor de búsqueda: Interfaz de Google con fines educativos

## Descripción
> Motor de búsqueda inspirado en la experiencia de Google: una recreación enfocada en fidelidad visual, diseño adaptable y arquitectura de componentes reutilizables. Desarrollado con fines educativos y complementarios al curso HTML5 de EDUTIN.
>
> Enlace público (DEMO): [Motor de Búsqueda by Odra Sanchez](https://frontend-motor-busqueda.netlify.app/)

## Cómo ejecutar el proyecto 🚀

1. Clona el repositorio: 
   git clone https://github.com/odrasanchezdev/project-frontend-motor-busqueda.git

2. Abre el archivo `index.html` en tu navegador.

   También puedes utilizar la extensión **Live Server** de Visual Studio Code para una mejor experiencia durante el desarrollo.
